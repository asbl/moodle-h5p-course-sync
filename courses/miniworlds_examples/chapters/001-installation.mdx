### Das Ziel

Du lernst, wie du miniworlds auf deinem Computer installierst.

### Installation mit pip

```bash
pip install miniworlds
```

### Optional: Physics-Paket

Für Physik-Unterstützung (Schwerkraft, Kollisionen):

```bash
pip install miniworlds miniworlds_physics
```
