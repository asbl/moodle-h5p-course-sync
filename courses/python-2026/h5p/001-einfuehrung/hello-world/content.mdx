```python
print("hello World")
```

Wenn du den Text zwischen den Anführungszeichen änderst, 
siehst du, dass das Programm eine unterschiedliche Ausgabe gibt.

Was du siehst ist ein Programm mit einer **Anweisung**:

Es wird die Funktion **print** aufgerufen und der Funktion wird als
Parameter eine **Zeichenkette** (Engl: 'string') übergeben.

Die Funktion gibt dann diese Zeichenkette auf der Konsole aus.
