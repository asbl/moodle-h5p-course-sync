Eine gute Funktion löst eine klar benannte Teilaufgabe. Der Name sollte
verraten, was die Funktion macht.

```python
def begruessung(name):
    print("Hallo", name)


begruessung("Alex")
begruessung("Sam")
```

Die Funktion `begruessung` kann mit verschiedenen Namen aufgerufen werden.
Der Code wird nicht kopiert, sondern wiederverwendet.

### Funktionen planen

Vor dem Programmieren hilft eine kurze Planung:

```python noeditor
Name:        baue_statuszeile
Parameter:   name, leben, brot
Rückgabe:    ein fertiger Text
Beispiel:    baue_statuszeile("Alex", 8, 3)
             -> "Alex | Leben: 8 | Brot: 3"
```

### Aufgabe

Lies die Planung und ergänze die Funktion.

```python
def baue_statuszeile(name, leben, brot):
    # Ergänze die Rückgabe.

print(baue_statuszeile("Alex", 8, 3))
```
