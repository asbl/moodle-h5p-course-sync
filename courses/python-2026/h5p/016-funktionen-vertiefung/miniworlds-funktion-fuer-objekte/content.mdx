In miniworlds entstehen schnell mehrere ähnliche Objekte. Dafür sind Funktionen
ideal.

```python
import miniworlds

world = miniworlds.World(560, 300)
world.add_background((229, 238, 230))

def block(x, y, farbe):
    actor = miniworlds.Actor((x, y))
    actor.size = (45, 45)
    actor.color = farbe
    return actor


block(120, 220, (120, 90, 60))
block(170, 220, (120, 90, 60))
block(220, 220, (90, 150, 95))
block(270, 220, (90, 150, 95))

world.run()
```

Die Funktion `block` erzeugt ein Objekt und gibt es zurück. Dadurch kannst du
das Objekt später noch speichern oder verändern.
