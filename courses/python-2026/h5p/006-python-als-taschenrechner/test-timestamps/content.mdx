<Instructions>
Zeiten werden im Computer intern als Timestamp ("Zeitstempel") gespeichert. Ein Timestamp speichert die Anzahl der Sekunden, die seit 1970 vergangen ist.

Lese zwei timestamps ein und berechne die Anzahl an Sekunden zwischen zwei Timestamps

#### Beispiel:

* timestamp1: 1515142800  # 05.01.2018 10:00
* timestamp2: 1515142860  # 05.01.2018 10:01

Ergebnis: 60

* timestamp1: 0           # 01.01.1970 01:00 
* timestamp2: 1515142800  # 05.01.2018 10:00

Ergebnis: 1515142800

!!! note Betrag
Verwende die Funktion ```abs```, um den Betrag einer Zahl zu berechnen.
```abs(-3) # = 3```
!!!

!!! note Eingabe
Ihr müsst hier die Eingabe mit input selbst schreiben!
!!!
