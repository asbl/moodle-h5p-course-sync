### Operatoren

Mit den Operatoren `+, - , * , /` kannst du die Grundrechenarten verwenden.

Es gilt die übliche Reihenfolge der Operatoren (z.B. Punkt vor Strich) 
und du kannst auch Klammern setzen.

```python
print(6 * 3 +(1 + 1))
print((3 * 5) / (2 + 3))
```

### Potenzen

Potenzen verwendest du mit dem Operator `**`:

```python
print(2 ** 8)
print(3 ** 4)
```

### Ein Hinweis zur Schreibweise:

Üblicherweise setzt man vor und nach einem 
Operator ein Leerzeichen zur besseren Lesbarkeit,
also z.B.

```python

python
```

dein Programm besser lesbar wird.
