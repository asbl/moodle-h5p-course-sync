Hier siehst du ein erstes Turtle-Grafik-Programm. 

Turtle-Grafik basiert auf einer einfach zu verstehenden Metapher:

Stell dir vor, du hast eine Schildkröte, 
die von oben betrachtet wird und englische Befehle versteht. 
Der Schwanz der Schildkröte berührt den Boden und zeichnet dabei eine Spur 
auf den Boden.

So sieht das erste Programm in Turtle-Grafik aus:

```python
import turtle
turtle.forward(100)
turtle.right(90)
```

## Wie ist das Programm aufgebaut?

- Zeile 1: import turtle - Importiert die Turtle-Grafik Bibliothek
- turtle.forward(100) - Die Turtle bewegt sich von ihrer Ausgangsposition 100 Schritte vorwärts und zeichnet dabei einen Strich
- turtle.right(90) - Die Turtle dreht sich um 90° nach rechts.

### Jetzt bist du dran: 

Verändere dieses Programm! 
Du kannst die Schrittweite verändern, 
die Gradzahl bei der Drehung und kannst weitere Zeilen hinzufügen. 

Versuche die folgenden Bilder zu zeichnen:

```python
import turtle
turtle.forward(100)
turtle.right(90)
```

!!! note 
Man kann die Turtle auch in eine echte Schuldkröte verwandeln
!!!

```python
import turtle

turtle.shape("turtle")
turtle.forward(100)
```
