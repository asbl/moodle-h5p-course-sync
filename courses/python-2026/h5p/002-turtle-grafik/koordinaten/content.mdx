Turtle-Grafik verwendet ein Zeichenfenster und strukturiert dieses
mit Hilfe eines Koordinatensystems


Du kannst auf dieses Koordinatensystem direkt zugreifen mit
folgendem Befehl:

```python
turtle.goto(x, y) # (Setze für x und y Zahlen ein)
```

Die Position (0,0) bezeichnet dabei den Ursprung. 

Die erste Koordinate bezeichnet immer die x-Achse, die zweite Koordinate
die y-Achse. Beide Koordinaten können auch negative Werte haben.

Außerdem kann man die Turtle auch direkt in eine bestimmte Richtung drehen. 
Man verwendet dafür den folgenden Befehl:

```python
setheading([winkel])
```

Man setzt für `[Winkel]` einen Winkel als Zahl ein, z.B. `setheading(120)` 
oder als Dezimalzahl `setheading(120.0)` (Beachte, dass man hier
wie in der englischen Sprache ein . statt eines Kommas verwendet).)

Zur Orientierung: Den Winkeln kann man folgende Richtungen zuordnen:

  * 0: Osten 
  * 90 Norden 
  * 180: Westen 
  * 270: Osten
