Probiere selbst. Zeichne eine Verkehrsampel

```python
import turtle

turtle.left(90)
turtle.pensize(60)
turtle.forward(120)
turtle.color(1,0,0)
turtle.dot(40)

turtle.penup()
turtle.back(50)
turtle.pendown()
turtle.color(1,1,0)
turtle.dot(40)
```

Verwende folgende Anweisungen:

```python
turtle.dot(n) # Zeichnet einen Punkt mit Durchmesser 20
turtle.pendown() # Hebt den Stift an. Du kannst die turtle nun bewegen ohne zu zeichnen
turtle.penup() # Nun zeichnet die turtle wieder
turtle.back(n) # Bewegt die Turtle um n Schritte rückwärts
```
