### Übung 5 – Beurteilen

Beide Funktionen sollen prüfen, ob **alle** Zahlen in der Liste positiv sind.
Welche ist besser geeignet? Begründe.

```python
def alle_positiv_a(zahlen):
    for z in zahlen:
        if z <= 0:
            return False
    return True

def alle_positiv_b(zahlen):
    positiv = 0
    for z in zahlen:
        if z > 0:
            positiv = positiv + 1
    return positiv == len(zahlen)
```
