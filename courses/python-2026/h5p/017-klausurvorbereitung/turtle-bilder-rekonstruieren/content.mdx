Bei Turtle-Aufgaben lohnt sich ein ruhiges Vorgehen:

1. Bestimme den Startpunkt.
2. Markiere Richtungen und Drehwinkel.
3. Suche Wiederholungen.
4. Schreibe erst danach den Code.

### Übung

Der folgende Code zeichnet drei gleich große Quadrate nebeneinander.

```python
import turtle

turtle.speed(0)
turtle.penup()
turtle.goto(-120, -40)
turtle.pendown()

for block in range(3):
    for i in range(4):
        turtle.forward(60)
        turtle.left(90)
    turtle.penup()
    turtle.forward(80)
    turtle.pendown()
```

Ändere ihn so, dass statt drei Quadraten vier Quadrate entstehen und jedes
zweite Quadrat eine andere Füllfarbe bekommt.
