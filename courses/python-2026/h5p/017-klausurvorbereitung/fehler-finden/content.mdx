### Block C

Die Funktion soll `True` zurückgeben, wenn `anzahl` **mindestens** 3 ist.
Für `anzahl = 3` gibt sie aber `False` zurück. Was ist falsch?

```python
def mindestens_drei(anzahl):
    return anzahl > 3

print(mindestens_drei(5))   # True
```

### Block E – Funktion ohne Wirkung

Die Funktion soll die Summe aller Listenelemente berechnen, gibt aber
immer `0` zurück. Was ist falsch?

```python
def summe(liste):
    gesamt = 0
    for z in liste:
        gesamt + z
    return gesamt

print(summe([1, 2, 3, 4]))  # soll 10 ergeben
```

### Code-Review

Beurteile den folgenden Code. Was funktioniert nicht wie erwartet? Erkläre
das Problem und schreibe eine korrigierte Version.

```python
def alle_positiv(liste):
    ergebnis = False
    for element in liste:
        if element > 0:
            ergebnis = True
        else:
            ergebnis = False
    return ergebnis

print(alle_positiv([1, 2, 3]))    # soll True sein
print(alle_positiv([-1, 2, 3]))   # soll False sein
```

Teste beide Aufrufe, erkläre dann: Wann gibt die Funktion ein falsches
Ergebnis, und warum?
