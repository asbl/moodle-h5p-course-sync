In Python ist es nicht möglich leere Blöcke zu schreiben, wenn nichts getan werden soll.
Beispiel:

Es soll nichts getan werden, wenn x>0 ist. Ansonsten soll x ausgegeben werden.

```python
if x > 0:
else:
    print(x)
```

Das Programm führt zu einem Fehler, da Python nach der Zeile 1 einen eingerückten Block erwartet. Ein Python Programm ist nämlich so aufgebaut, dass nach einem if-statement **immer* ein eingerückter Block folgt.
Die pass-Anweisung

Wenn in einem Block nichts getan werden soll, dann verwende die pass-Anweisung. Diese bedeutet so viel wie "tue nichts":
Beispiel

```python
x = int(input())

if x > 0:
   pass
else:
    print(x)
```

Jetzt ist der Block nach ```if``` korrekt eingerückt: 
Python wirft keinen Fehler mehr.
