Die If-Anweisung kann um einen Else-Block erweitert werden. Mit Hilfe von else 
kannst du festlegen, welche Bedingung ausgeführt wird, wenn die Bedingung 
**nicht** zutrifft. 

### Beispiel

```python
x = int(input("Gebe eine Zahl ein:")) 
if x % 2 == 0:  
    print("Die Zahl ist gerade")
else:
  print("Die Zahl ist ungerade")
```

Der Code bedeutet folgendes: 

* Wenn x gerade ist (d.h. sie ist ohne Rest durch 2 teilbar), dann wird ausgegeben, 
  dass die Zahl gerade ist.
* **Andernfalls* wird ausgegeben, 
  dass die Zahl ungerade ist.
