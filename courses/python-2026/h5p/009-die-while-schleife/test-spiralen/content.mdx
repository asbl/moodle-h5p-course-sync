### Beschreibung:
Zeichne eine Spirale aus Linien, die sich nach außen bewegen.

### Regeln:

Die Spirale startet in der Mitte (0, 0).

Die Länge der ersten Linie ist 25.

Jede nächste Linie ist um 5 länger als die vorherige.

Nach jeder Linie dreht sich die Figur 90° im Uhrzeigersinn.

Stoppe, wenn die Länge der Linie ≥ x ist (x wird mit input() eingelesen.

Verwende eine while-Schleife, weil die Anzahl der Linien vorher nicht bekannt ist.
</Instructions>

```python editor:preCode
import turtle
x = int(input())
```

```python grading:targetCode
import turtle

x = int(input())

t = turtle.Turtle()
t.speed(0)

length = 25
while length <= x:
    t.forward(length)
    t.right(90)
    length += 5

turtle.done()
```
