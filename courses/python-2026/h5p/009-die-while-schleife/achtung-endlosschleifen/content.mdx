Achtung: Mit while-Schleifen ist es möglich,
**Endlosschleifen** zu produzieren. Manchmal ist dies gewollt,
aber manchmal führt dies auch dazu, dass dein Programm Ewigkeiten an einer Stelle hängt.

```python
x = int(input())

while(x < 10):
    x = x - 1

# Dieser Code wird niemals erreicht.
print(x)
```
