Variablen sind das **"Gehirn des Computers"**.

In Variablen kann man Werte speichern und später wieder verwenden. 
Dies klingt nicht nach viel, ist aber eines der wichtigsten Konzept 
bei der Programmierung.

Eine erste Variablenzuweisung in Python kann so aussehen:

```python
age = 18
```

Diese Zuweisung besteht aus zwei Teilen. 

Links steht ein **Name**, rechts ein **Wert**

Die Zahl 18 wird irgendwo an einer Adresse im Computer gespeichert.  
Damit sie später wiedergefunden werden kann, bekommt sie den **Namen** `age`.

### Beispiel

```python
a = 3
b = 4
print(a + b)
```

Hier wurden die Zahlen 3 und 4 an den Speicherorten a und b gespeichert.
Die Print-Anweisung kann anschließend auf die beiden Werte zugreifen.
