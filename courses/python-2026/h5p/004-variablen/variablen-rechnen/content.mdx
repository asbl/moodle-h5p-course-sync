Man kann mit Variablen genauso wie mit Zahlen rechnen:

```python
a = 3
b = 4
a = a + b
```

Auch hier spielt das **Benennen von Variablen** eine besondere Bedeutung. 
Betrachte z.B. die folgenden beiden Codes:

```python
a = 8
b = 10.45
print(a * b)
```

und

```python
stunden = 8
lohn_pro_stunde = 10.45
print(stunden * lohn_pro_stunde)
```

**Semantisch** unterscheiden sich die beiden Programme nicht. 
Es wird dasselbe Ergebnis ausgegeben. Das zweite Programm ist aber besser lesbar.
