Mit `+` kannst du zwei Strings verbinden. Wenn du aber mehrere Textteile
zusammensetzen möchtest, wird das schnell unübersichtlich. 

Für solche Fälle gibt es `join()`.

Beispiel:

```python
woerter = ["laterne", "kompass", "notiz"]

print(", ".join(woerter)) # Die Wörter werden mit , verbunden
print(" - ".join(woerter)) # ...oder mit Bindestrich
```

Der String vor `.join()` ist der Trenner. Er wird zwischen die einzelnen
Elemente gesetzt.

```python noeditor
", ".join(["laterne", "kompass", "notiz"])

laterne + ", " + kompass + ", " + notiz
```

!!! note Wichtig
`join()` erwartet eine Liste von **Strings**. Wenn in der Liste Zahlen liegen,
musst du sie vorher in Strings umwandeln.
!!!

### Warum ist die join() Funktion hilfreich?

Stell dir vor, du möchtest eine Liste von Gegenständen ausgeben:

```python noeditor
inventar = ["Schlüssel", "Funkgerät", "Pausenbrot"]

text = "Inventar: " + ", ".join(inventar)
print(text)
```

Das ist besser als:

```python noeditor
print(inventar[0] + ", " + inventar[1] + ", " + inventar[2])
```

Die zweite Variante funktioniert nur für genau drei Einträge. `join()` passt
viel besser zu Listen.

### Exkurs: split() und join()

`split()` und `join()` sind Gegenstücke. 

* `split()` macht aus einem String eine Liste. 
* `join()` macht aus einer Liste wieder einen String.

```python
satz = "Runen leuchten leise"
woerter = satz.split()

print(woerter)
print("|".join(woerter)) # Gibt wieder den ursprünglichen Satz aus.
```

Aus `"Runen leuchten leise"` wird zuerst:

```python
["Runen", "leuchten", "leise"]
```

Danach verbindet `"|" .join(...)` die Wörter mit einem senkrechten Strich.

```python noeditor
"Runen leuchten leise"
        split()
["Runen", "leuchten", "leise"]
        "|".join(...)
"Runen|leuchten|leise"
```

### Kombination mit f-Strings

`join()` und f-Strings passen gut zusammen:

```python
name = "Taro"
hinweise = ["Spind 13", "blauer Würfel", "leerer Korridor"]

print(f"{name} notiert: {', '.join(hinweise)}")
```
